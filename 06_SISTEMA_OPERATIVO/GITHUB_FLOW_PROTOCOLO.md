# ⛓️ Protocolo GitHub Flow v4.3

El Will-AI Project Lab adopta una metodología de despliegue continuo basada en GitHub Flow para asegurar la agilidad operativa.

## 1. Rama Principal (main)
La rama `main` contiene el estado consolidado y verificado de la Arquitectura Integral v4.3.

## 2. Ramas de Trabajo (feature/fix)
Cualquier cambio estructural o nuevo proyecto debe iniciarse en una rama separada para su validación previa.

## 3. Pull Requests (PR)
Zara ejecuta y Sylvia Bloom supervisa. No se integra nada en `main` sin la validación del Vórtice de 180 grados.

## 4. Automatización
Se prioriza el uso de la API de GitHub para la gestión de activos y respaldos, garantizando la integridad de los datos.
